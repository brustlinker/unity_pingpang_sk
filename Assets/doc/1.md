### 执行顺序

```
if(Input.GetKey(upKey))
{
    rigid2d.velocity=new Vector2(0,speed);
}
else if(Input.GetKey(downKey))
{
    rigid2d.velocity=new Vector2(0,-speed);
}
else
{
    rigid2d.velocity=new Vector2(0,0);
}
```

这个的执行顺序是，如果上下键同时摁下，那么只有上面的响应。
